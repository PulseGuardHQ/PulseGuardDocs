## Overview

The Toolbox provides various utilities for diagnosing and analyzing your web resources. These tools help you troubleshoot issues, optimize performance, and ensure security compliance.

<Note>
  Full access to all toolbox features is available on Pro and Expert plans. Free accounts have limited access to select tools.
</Note>

## Using the DNS Lookup Tool

The DNS Lookup tool allows you to check various DNS record types for any domain:

<Steps>
  <Step title="Navigate to the Toolbox">
    Click on "Toolbox" in the main sidebar navigation
  </Step>
  <Step title="Select DNS Lookup">
    Click on the DNS Lookup tool card
  </Step>
  <Step title="Enter domain name">
    Enter the domain you want to check (e.g., example.com)
  </Step>
  <Step title="Select record types">
    Choose which DNS record types to query (or select "All")
  </Step>
  <Step title="View results">
    Review the DNS records displayed in the results panel
  </Step>
</Steps>

## SSL Checker

The SSL Checker analyzes the SSL/TLS certificate of a website to verify its validity, security, and configuration:

<AccordionGroup>
  <Accordion title="Certificate Details">
    - Issuer information
    - Valid from/to dates
    - Certificate chain
    - Subject Alternative Names (SANs)
  </Accordion>
  <Accordion title="Security Analysis">
    - SSL/TLS protocol versions
    - Cipher suites
    - Key strength
    - Vulnerability checks (BEAST, POODLE, Heartbleed)
  </Accordion>
  <Accordion title="Configuration Check">
    - OCSP stapling
    - HSTS implementation
    - Certificate transparency
    - Trust chain verification
  </Accordion>
</AccordionGroup>

## Port Scanner

The Port Scanner tool checks for open ports on a server or IP address:

<Steps>
  <Step title="Navigate to the Toolbox">
    Click on "Toolbox" in the main sidebar navigation
  </Step>
  <Step title="Select Port Scanner">
    Click on the Port Scanner tool card
  </Step>
  <Step title="Enter target">
    Enter the domain or IP address you want to scan
  </Step>
  <Step title="Select ports">
    Choose common ports or specify custom port ranges
  </Step>
  <Step title="Start scan">
    Click the "Start Scan" button to begin the port scan
  </Step>
</Steps>

<Note>
  The Port Scanner is only available on Pro and Expert plans. Use this tool responsibly and only scan servers you have permission to test.
</Note>

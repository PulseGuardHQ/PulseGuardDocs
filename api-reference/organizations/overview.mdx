The Organizations API enables team collaboration, resource sharing, and role-based access control for Enterprise customers.

## Features

- **Team Management** - Invite and manage team members
- **Resource Sharing** - Share domains, devices, and services across the organization
- **Role-Based Access** - Granular permissions for different team roles
- **Unified Billing** - Centralized subscription management
- **Activity Tracking** - Audit logs for team actions

## Plan Requirements

Organizations are **Enterprise Plan only**:
- **Enterprise Plan** - Full organization features
- **Other Plans** - Organization features not available

## Authentication

Organization endpoints require:
- **Sanctum Authentication** - Valid API token
- **Enterprise Plan** - Verified via `enterprise` middleware
- **Organization Access** - User must belong to the organization

## Team Roles

Organizations support these roles:

- **Owner** - Full organization control
- **Admin** - Manage resources and members
- **Member** - Access shared resources
- **Viewer** - Read-only access

## Resource Sharing

Organizations can share:

### Domains
- Domain monitoring access
- DNS management permissions
- Alert notifications

### Devices  
- Device monitoring data
- Command execution rights
- Metrics and logs access

### Services
- Service monitor access
- Configuration management
- Alert routing

## Common Response Format

```json
{
  "success": true,
  "message": "Operation completed successfully",
  "data": {
    "organization": {
      "id": 1,
      "name": "Acme Corp",
      "slug": "acme-corp",
      "members_count": 5,
      "created_at": "2024-01-15T10:30:00Z"
    }
  }
}
```

## Invitation Flow

1. **Send Invitation** - Owner/Admin invites user by email
2. **Email Notification** - User receives invitation link
3. **Accept Invitation** - User joins organization
4. **Role Assignment** - User granted appropriate permissions

## Limitations

Enterprise organizations have generous limits:
- **Members** - Up to 50 team members
- **Shared Resources** - Unlimited sharing within organization
- **API Calls** - Higher rate limits for organization endpoints

---
title: 'Update Device'
method: 'PUT'
path: '/api/v1/devices/{uuid}'
---

This endpoint allows you to update an existing device.

### Path Parameters

<ParamField path="uuid" type="string" required>
  The universally unique identifier of the device.
</ParamField>

### Body

You can send the same parameters as when creating a device to update its values.

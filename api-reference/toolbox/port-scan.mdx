This endpoint performs a port scan on a domain or IP address.

### Body

<ParamField body="domain" type="string" required>
  The domain or IP address to scan.
</ParamField>
<ParamField body="ports" type="string">
  A comma-separated list of ports to scan (e.g., `80,443,3306`). Defaults to common ports.
</ParamField>

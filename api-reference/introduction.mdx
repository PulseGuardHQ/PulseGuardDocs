# PulseGuard API

Welcome to the PulseGuard API! Our comprehensive REST API allows you to programmatically manage your monitoring infrastructure, retrieve analytics data, and integrate PulseGuard with your existing tools and workflows.

## API Versions

PulseGuard supports multiple API versions:

- **v2** - Legacy API (maintained for backward compatibility)  

## Base URLs

```

# v2 API (Legacy)  
https://app.pulseguard.nl/api/v2/
```

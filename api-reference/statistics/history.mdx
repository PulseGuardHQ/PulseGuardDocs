Retrieves response time history for a domain.

### Path Parameters

<ParamField path="uuid" type="string" required>
  The universally unique identifier of the domain.
</ParamField>

### Query Parameters

<ParamField query="range" type="string">
  Time range for data (`24h`, `7d`, `30d`, `all`). Default: `24h`.
</ParamField>
